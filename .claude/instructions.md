# Claude Instructions for Florecer Mind

## Before Creating ANY New Content

**ALWAYS read `/BRAND_GUIDE.md` first** before creating:
- New articles
- New components
- UI elements
- Marketing copy
- Social media content

## Article Creation Workflow

1. **Read the brand guide**: `BRAND_GUIDE.md`
2. **Use the template**: `.templates/article-template.md`
3. **Follow the structure**: Opening hook → Research → Explanation → Practical Applications → For Counselors (if applicable) → Key Takeaways → Getting Started
4. **Verify against checklist**: See "Publication Checklist" in brand guide
5. **Cite all sources**: Include in frontmatter with URLs

## Component Creation Workflow

1. **Read the brand guide**: Component patterns section
2. **Use design tokens**: Colors, spacing, typography from brand guide
3. **Follow button patterns**: `.btn` classes with proper variants
4. **Ensure accessibility**: ARIA labels, semantic HTML, focus states
5. **Test responsiveness**: Mobile-first, consistent breakpoints

## Quality Standards

**Never publish without:**
- ✓ Research backing for all claims
- ✓ Brand colors and typography
- ✓ Proper spacing and formatting
- ✓ Accessibility considerations
- ✓ Mobile responsiveness
- ✓ Proofreading for clarity

## Key Reminders

- Voice: Warm but authoritative, evidence-based but accessible
- No jargon without explanation
- Checkmarks (✓) for research findings
- Citations: (Year, N=sample size) format
- Font: Crimson Pro for headings, Inter for body
- Colors: #2F5233 (primary), #A8B5A0 (secondary)
- Transitions: 300ms standard
- Spacing: Use 4px base unit

## When In Doubt

Refer to the brand guide. Consistency > creativity. Quality > speed.
